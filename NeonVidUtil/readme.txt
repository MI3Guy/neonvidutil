Neon VidUtil

Neon VidUtil is an open source video processing utility.


Neon VidUtil uses the following libraries and utilities.

Wav2Flac - Stanimir Stoyanov (http://stoyanoff.info/blog/2010/01/08/encoding-uncompressed-audio-with-flac-in-c/)
libFLAC - (http://flac.sourceforge.net/api/)
libMediaInfo - (http://mediainfo.sourceforge.net/)
mkvtoolnix - (http://www.bunkus.org/videotools/mkvtoolnix/)
vc1conv - (http://w6rz.net/)
